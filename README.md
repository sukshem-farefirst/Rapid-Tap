# Click Speed Gam

A simple and fun **Click Speed Game** built with **React** and **React Router**, where players try to click as many times as possible within **10 seconds**. Scores are stored in the browser and displayed in a leaderboard.

---

## Features

- Enter player name to start the game
- 10-second countdown timer
- Real-time click counter
- Store scores in **localStorage**
- Leaderboard with ranking
- Navigate between **Game** and **Leaderboard**

---

## Setup

- `node 22`
- `react 19`
